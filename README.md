# Simple Calculator

This is a simple calculator application that is easy to use and can assist you in performing basic mathematical calculations. This application is created using the Swift programming language.

## Features:

- Addition
- Subtraction
- Multiplication
- Division
- Clear & Clear All
- Percentage
